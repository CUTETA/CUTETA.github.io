Tanner Install

一、以管理员身份运行setup.exe

二、选择 Local license

接下来一路默认

三、选择 Install

继续一路默认

User name 和 Organization 随便填写即可

接下来设置安装路径，注意：安装路径不可有中文！！

继续默认

点击 Install 开始安装

四、安装 Tanner EDA Utilites

一路Next，安装步骤参考上一环节

注意安装路径不要有中文！！！

继续默认

五、安装 System Driver

一路默认 Next 即可

接着选择本地许可

六、解压 patch.rar

进入 Crack 文件夹

解压 patch.rar 并将解压出来的文件复制粘贴至安装目录(我这里是D:\Tanner EDA\Tanner Tools v16.0)
